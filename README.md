# 📝 FastAPI Todo Application
A production-ready Todo management application with user authentication, WhatsApp notifications, and beautiful UI.
## ✨ Features
- 🔐 **User Authentication** - Secure JWT-based authentication
- 📱 **WhatsApp Notifications** - Get reminders 10 minutes before tasks
- 🎨 **Beautiful UI** - Responsive design with modern aesthetics
- ⚡ **Fast API** - Built with FastAPI for high performance
- 🗃️ **Database Support** - Works with MySQL (local) and PostgreSQL (production)
- ✅ **Task Management** - Full CRUD operations with priority levels
- 📊 **User Dashboard** - Track all your tasks in one place
- 🧪 **Tested** - Comprehensive test coverage with pytest
## 🚀 Quick Start
Visit http://127.0.0.1:8000 after starting the server.
## 📱 WhatsApp Setup (Optional)
Get free Twilio account: https://www.twilio.com/try-twilio
## 🛠️ Tech Stack
- FastAPI, SQLAlchemy, MySQL/PostgreSQL
- JWT authentication with bcrypt
- Twilio WhatsApp API for notifications
